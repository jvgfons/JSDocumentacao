Introdução

JavaScript é uma linguagem de programação utilizada na web para tornar páginas interativas.

É interpretada, orientada a objetos e baseada em protótipos.

JavaScript pode ser executado em navegadores ou no servidor com Node.js.

console.log('Olá, mundo!');
  • Interatividade
  • Execução no navegador
  • Popularidade alta
Variáveis e Tipos

Existem três formas de declarar variáveis: var, let e const.

Tipos primitivos incluem string, number, boolean, null, undefined e symbol.

O tipo de uma variável pode ser verificado com typeof.

let nome = "João";
  • var é função-local
  • let e const são bloco-local
  • const não permite reatribuição
Funções

Funções são blocos de código que executam tarefas quando chamadas.

Podem ser declaradas com a palavra-chave function ou como funções anônimas.

Funções também podem ser atribuídas a variáveis ou passadas como argumento.

function soma(a, b) { return a + b; }
  • Funções nomeadas
  • Funções anônimas
  • Arrow functions
Objetos

Objetos são coleções de pares chave-valor.

Podem conter métodos, que são funções dentro de objetos.

É possível acessar propriedades com notação ponto ou colchete.

let pessoa = { nome: "Ana", idade: 25 };
  • Flexíveis
  • Estruturam dados
  • Suportam herança via protótipos
Eventos

Eventos permitem responder a ações do usuário, como cliques ou teclas pressionadas.

São tratados com addEventListener.

Podem ser associados a qualquer elemento do DOM.

botao.addEventListener("click", function() { alert("Clicou!"); });
  • click
  • keydown
  • submit
